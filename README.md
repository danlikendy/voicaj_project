## О проекте

**Voicaj** — это инновационное iOS-приложение, которое превращает голосовые заметки в структурированные задачи и планы. Используя передовые технологии распознавания речи и искусственного интеллекта, приложение помогает пользователям эффективно организовывать свою жизнь через естественное голосовое взаимодействие

### Ключевые особенности

- **Голосовое создание задач** — говорите, а приложение создает структурированные задачи
- **AI-анализ контента** — автоматическое определение приоритетов, сроков и категорий
- **Адаптивный дизайн** — интерфейс подстраивается под время суток, погоду и настроение
- **Умная аналитика** — отслеживание прогресса и продуктивности
- **Автоматическая синхронизация** — ваши данные всегда актуальны
- **Темная тема** — поддержка светлой и темной темы с красивой палитрой цветов

## Быстрый старт

### Требования

- **Xcode**: 16.0+
- **iOS**: 18.5+
- **Swift**: 5.9+
- **macOS**: 14.0+ (для разработки)

### Установка

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/luvgreyair/voicaj_project.git
   cd voicaj_project
   ```

2. **Откройте проект в Xcode**
   ```bash
   open voicaj_project.xcodeproj
   ```

3. **Настройте подписание**
   - Выберите ваш Team в Signing & Capabilities
   - Убедитесь, что Bundle Identifier уникален

4. **Запустите приложение**
   - Выберите симулятор iPhone 16 или реальное устройство
   - Нажмите Run

## Основные функции

### Голосовое управление

- **Запись задач** — создавайте задачи простым голосовым описанием
- **Шаблоны** — используйте предустановленные шаблоны для быстрого создания
- **AI-анализ** — автоматическое определение приоритетов и категорий
- **Распознавание речи** — поддержка русского и английского языков

### Управление задачами

- **Статусы**: В планах, В работе, Завершено, На паузе
- **Приоритеты**: Высокий, Средний, Низкий
- **Теги и категории** — гибкая система классификации
- **Сроки и напоминания** — никогда не пропустите важное

### Аналитика и отчеты

- **Прогресс дня** — что выполнено и что в планах
- **Статистика недели** — анализ продуктивности
- **Тренды** — отслеживание изменений во времени
- **Экспорт** — поделитесь результатами с командой

### Анализ голоса

- **Определение настроения** — анализ тона и интонации
- **Извлечение сущностей** — автоматическое выделение дат, имен, мест
- **Классификация контента** — умное определение типа задачи
- **Персонализация** — адаптация под стиль пользователя

### Умные рекомендации

- **Оптимальное время** — когда лучше выполнять задачи
- **Группировка** — объединение связанных задач
- **Приоритизация** — автоматическое ранжирование по важности
- **Напоминания** — умные уведомления в нужный момент

## Экраны приложения

### Главный экран (Home)
- Обзор задач по статусам
- Быстрые действия с шаблонами
- Голосовая запись в один клик
- Фильтрация по хештегам и категориям

### Запись голоса (Recording)
- Простой интерфейс записи
- Визуализация аудиоволны
- Редактирование и перезапись
- Предварительный анализ

### Календарь (Calendar)
- Месячный и недельный вид
- Цветовая индикация статусов
- Drag & Drop для изменения сроков
- Интеграция с системным календарем

### Аналитика (Analytics)
- Графики продуктивности
- Статистика по категориям
- Сравнение периодов
- Экспорт отчетов
